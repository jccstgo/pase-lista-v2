ℹ️ Información sobre las Restricciones

Las restricciones permiten controlar dónde y cómo se puede registrar la asistencia. Los supervisores con claves administrativas pueden saltarse estas restricciones.

Solo permite registrar asistencia dentro de un radio específico de una ubicación.

Solo permite un registro por dispositivo/teléfono. Evita que múltiples personas usen el mismo dispositivo.

Permite que supervisores con claves especiales puedan saltarse las restricciones de ubicación y dispositivo.

📤 Subir Lista de Personal

📋 Formato Requerido:

matricula,nombre,grupo

A654321,Sargento María García Silva,b
B678904,Capitán 1/o Carlos López Martínez,c

Obligatorios: matricula, nombre, grupo

🧹 Limpiar Base de Datos

Elimina todos los registros de personal almacenados en la base de datos y reinicia la lista de asistencia y dispositivos registrados.

Esta acción no se puede deshacer. Asegúrate de contar con un respaldo antes de continuar.
